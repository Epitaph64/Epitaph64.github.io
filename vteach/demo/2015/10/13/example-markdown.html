A Brief Demonstration of the Markdown Markup Language


Oct 13, 2015 | Walter Macfarland

Categories: demo

Below (if you’re viewing the source of this document) is a demonstration of some of the Markdown syntax. If more fine-tuned control is required, the .markdown file containing the post can be renamed into a .html file, and the Markdown syntax will continue to work, as long as the YAML Front Matter (the content at the top of the document) remains intact.

Easily write a list of instructions:

Pedantic Lightbulb Replacement Instructions:

  1. Find the switch controlling the affected site and switch it off.
  2. Remove the destroyed bulb from its holder by unscrewing it in a series of counter-clockwise gentle rotations.
  3. Place a fresh bulb in the socket (reversing the movement used to remove the last bulb.)
  4. Reverse the instructions you followed after finding the switch in step 1.

A simple bulleted list.

  • Item A
  • Item B
  • Item C

A nested list structure:

  • Chess Trivia

Use ‘>’ to begin a blockquote element around the text which appears after. Leave a completely blank line to end the blockquote. To have text on multiple lines within a blockquote, simply use the <br> tag.
